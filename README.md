# Ayat

**Ayat** هو مشروع ويب بسيط يعرض آيات أو بيانات من القرآن الكريم باستخدام HTML وCSS وJavaScript.

---

## 📌 وصف المشروع
المشروع عبارة عن صفحة ويب ثابتة (Static) تقوم بعرض بيانات محفوظة في ملف `ayat.json`.  
الصفحة تستخدم ملفات CSS لتنسيق العرض وملفات JavaScript لتحميل وعرض البيانات في واجهة المستخدم.

---

## 📂 هيكل الملفات

```
Ayat/
├── index.html
├── css/
│   └── … (ملفات CSS لتنسيق الصفحة)
├── js/
│   └── … (ملفات JavaScript لعرض البيانات)
└── ayat.json
```

---

## 🧠 طريقة عمل المشروع

- **index.html**: الصفحة الرئيسية لعرض المحتوى.  
- **css/**: مجلد يحتوي على ملفات CSS لتصميم الواجهة.  
- **js/**: مجلد يحتوي على أكواد JavaScript لتحميل وعرض بيانات `ayat.json`.  
- **ayat.json**: ملف البيانات الذي يحتوي على الآيات أو المحتوى المراد عرضه.
